# Correlation and Factor Analysis

## Project Overview
This project applies correlation analysis and exploratory factor analysis to identify relationships among variables and detect underlying latent structures within a dataset.

## Analytical Process
The analysis includes:

- Data preparation and structuring in Excel
- Construction of a correlation matrix
- Identification of significant relationships among variables
- Factor extraction based on shared variance
- Interpretation of factor loadings to determine variable groupings

## Key Skills Demonstrated
- Quantitative data analysis
- Correlation analysis
- Exploratory factor analysis
- Dimensionality reduction
- Statistical reasoning
- Excel-based analytical modeling

## Files
- Factor_Analysis_Project.xlsx – Analytical workbook containing dataset, correlation matrix, and factor analysis calculations
- Factor_Analysis_Synopsis.pdf – One-page analytical explanation of the methodology and results
