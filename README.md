# Portfolio and Investment Analysis

## Project Overview

This project analyzes historical stock market data for two publicly traded companies obtained from Yahoo Finance. The objective is to evaluate stock performance, risk, and diversification benefits through quantitative analysis.

## Data and Methodology

Historical daily adjusted closing prices were collected and prepared in Microsoft Excel. Daily stock returns were calculated using percentage change between consecutive trading days. Key statistical measures including average return, standard deviation, and correlation were computed to evaluate asset performance and volatility.

A simple portfolio was constructed assuming equal allocation between the two assets. Daily portfolio returns were calculated to analyze overall portfolio performance and risk characteristics.

## Skills Demonstrated

* Financial data preparation and cleaning
* Quantitative analysis using Excel
* Return and risk measurement
* Correlation analysis and portfolio evaluation
* Data visualization and analytical reporting
